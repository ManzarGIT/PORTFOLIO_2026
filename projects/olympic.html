Olympic Data Analysis

Analyzing medal tallies using Python, Pandas & NumPy.

1. Challenge & Data

Used athlete_events.csv and noc_regions.csv to examine Olympic performance trends across countries and years.

2. Data Preparation

  • Handled missing values and standardized country names.
  • Removed duplicates and added new metrics (total medals, gold ratio).

3. Analysis

Grouped by year and country to rank nations, find top medalists, and chart medal trends. USA dominated overall; host nations often outperformed average.

4. Outcome

Pandas + NumPy delivered fast medal analysis and Matplotlib visualizations—bar charts, line plots, stacked graphs—highlighting historical trends.
